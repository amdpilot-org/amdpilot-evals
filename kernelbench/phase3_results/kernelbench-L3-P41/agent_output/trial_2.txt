  • Each layer has forward and reverse directions
  • Each direction has: weight_ih (input-to-hidden: 3hidden x input), weight_hh

  • The 3*hidden comes from the 3 gates: reset (r), update (z), and new (n)

  The GRU equations per timestep:

  r = sigmoid(W_ir @ x + W_hr @ h_prev + b_ir + b_hr)
  z = sigmoid(W_iz @ x + W_hz @ h_prev + b_iz + b_hz)
  n = tanh(W_in @ x + b_in + r * (W_hn @ h_prev + b_hn))
  h_new = (1 - z) * n + z * h_prev
